Purpose and Information

Site Purpose:

"Is It Walking Weather?" provides short-term forecasting data to help people decide when to go out in each town. It simplifies the data other sources might provide into windows of enjoyable weather based on precipitation, temperature, humidity, etc.

If there isn't a window of comfortable weather, the site will state that today isn't good for walking, but will still provide the best time window available, so long as a safe one exists.

The site will also feature news about local events that one might run into while walking about, such as parades or races.
